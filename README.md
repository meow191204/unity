### <게임 진행 방법><br>

1. Start 버튼을 눌러 게임 시작<br>

2. NPC가 주는 퀘스트를 보고 인공지능소프트웨어학과와 관련된 실을 찾아감<br>

3. 잘못된 실에 들어가면 처음부터 재시작됨<br>

4. 마지막 실에 들어가면 NPC가 게임 클리어를 알리고 Retry 버튼이 등장함 <br>
